# 天玑 · 商业活动全链路平台审计

> 审计逻辑：一个跨境电商业者（Furry Baby）从产品到售后，需要看什么平台？

---

## 商业活动七环节 × 覆盖度

```
产品研发 → 采购生产 → 营销推广 → 销售运营 → 物流交付 → 售后服务 → 财税合规
```

| 环节 | 天玑覆盖度 | 核心缺口 |
|------|-----------|---------|
| ① 产品研发/选品 | ✅ 80% | 社媒选品、竞品分析、认证要求都有 |
| ② 采购/供应链 | ⚠️ 50% | **缺产能监控、缺工厂评估、缺BOM成本追踪** |
| ③ 营销推广 | ❌ 20% | **缺广告平台、缺红人工具、缺SEO数据** |
| ④ 销售运营 | ⚠️ 60% | 电商平台有，**缺平台规则变更主动推送** |
| ⑤ 物流交付 | ❌ 10% | **缺快递API、缺海外仓状态、缺清关状态** |
| ⑥ 售后服务 | ❌ 5% | **缺退货分析、缺纠纷数据、缺客户反馈聚合** |
| ⑦ 财税合规 | ⚠️ 50% | 公司状态有，**缺销售税/VAT申报工具、缺成本核算** |

---

## 具体缺什么

### 1. 营销推广 — 最大缺口

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **Google Ads** | 你花广告费的地方 | 竞品投放策略、关键词CPC变化、政策更新 |
| **Meta Ads (FB/IG)** | 你花广告费的地方 | 广告政策变更、受众变化、表情审核规则 |
| **TikTok Ads** | 你花广告费的地方 | 广告审核规则、内容合规、投放趋势 |
| **TikTok Shop** | 直播带货渠道 | 政策变更、保证金调整、品类限制 |
| **Google Search Console** | SEO基础 | 关键词排名、流量来源、技术SEO问题 |
| **Ahrefs / SEMrush** | SEO竞品分析 | 竞品关键词、外链、内容策略 |
| **Amazon Advertising** | Amazon广告 | 竞品广告位、ACOS基准、品类趋势 |
| **红人平台 (Upfluence/Grin)** | 红人营销 | 红人数据、合作案例、ROI分析 |
| **联盟平台 (ShareASale/Impact)** | 联盟营销 | 佣金基准、联盟主趋势 |

### 2. 物流交付 — 直接影响客户体验

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **UPS / FedEx / DHL / USPS** | 你用的物流商 | 运费变动、服务变更、派送时效 |
| **中国邮政 / EMS** | 国内发货渠道 | 跨境小包政策 |
| **4PX / 云途 / 燕文** | 跨境专线 | 渠道时效、价格变动 |
| **CJ Dropshipping Dashboard** | 你正在用的 | 供应商状态、库存、异常发货 |
| **ShipStation / Shippo** | 多物流商聚合 | 比价、政策变更 |
| **港口状态 (LA/Long Beach/宁波)** | 海运周期 | 拥堵、延误、船期变化 |
| **清关行数据** | 清关时效 | 查验率、清关周期 |

### 3. 售后服务

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **Amazon Review API** | 竞品评论分析 | 差评关键词、功能缺陷、改进方向 |
| **Trustpilot** | 用户评价平台 | 品牌声誉监控 |
| **BBB (Better Business Bureau)** | 美国消费者投诉 | 投诉类型、处理标准 |
| **Shopify Order/Risk API** | 你自己订单 | 退款率、欺诈模式 |
| **退款/纠纷分析** | 售后优化 | 退货原因分布、成本归因 |

### 4. 财税合规

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **各州Sales Tax要求** | 美国税务合规 | 税率变更、阈值变化（Economic Nexus） |
| **VAT (英国/欧盟)** | UK/EU税务 | 注册门槛、税率变更 |
| **GST (加/澳/新/沙)** | 更多市场税务 | 注册要求 |
| **OFAC制裁清单** | 合规红线 | 受制裁实体/国家更新 |
| **FTC endorsement guides** | 营销合规 | 红人/推荐披露要求更新 |

### 5. 采购/供应链 — 更深入的

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **企查查/天眼查** | 工厂尽调 | 工商信息、司法风险、经营异常 |
| **1688指数** | 采购趋势 | 热门品、供应商活跃度 |
| **采购平台 (Global Sources)** | 新供应商发现 | 展会信息、工厂名录 |
| **BOM成本追踪** | 产品定价 | 每个零部件价格波动 |

### 6. 行业/协会 — 增长网络

| 缺的平台 | 为什么需要 | 数据价值 |
|---------|-----------|---------|
| **宠物用品协会 (APPA, WPA)** | 行业报告 | 市场规模、趋势、渠道分析 |
| **水处理协会 (WQA, IWA)** | 行业标准 | 技术趋势、行业活动 |
| **行业展会 (Global Pet Expo, ISH)** | 新品发现 | 新产品、新技术、新供应商 |
| **Nielsen / NPD** | 零售监测 | 品类销售趋势、渠道份额 |

## 覆盖度总结

| 评级 | 环节 | 覆盖 |
|------|------|------|
| ✅ 较好 | ①产品研发/选品 | ~80% |
| ⚠️ 中等 | ②采购供应链 | ~50% |
| ⚠️ 中等 | ④销售运营 | ~60% |
| ⚠️ 中等 | ⑦财税合规 | ~50% |
| ❌ 严重缺口 | ③营销推广 | ~20% |
| ❌ 严重缺口 | ⑤物流交付 | ~10% |
| ❌ 严重缺口 | ⑥售后服务 | ~5% |

**整体商业活动覆盖度：约40%**

## 下一步建议

按"对现金流的影响"排序，应该优先补什么：

| 优先级 | 环节 | 原因 |
|-------|------|------|
| **P0** | ③营销推广 | 你每个月花广告费，不知道平台政策变了就是烧钱 |
| **P0** | ⑤物流交付 | 运费涨了20%你的利润就没了，客户说没收到货你也不知道 |
| **P1** | ⑥售后服务 | 差评不处理，Listing权重一路掉 |
| **P1** | ⑦财税合规 | 美国Sales Tax漏申报罚款比税本身还多 |
| **P2** | ②采购供应链 | 你有供应商，扩展阶段才需要深挖 |
