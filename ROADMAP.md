## 贡献指南

天玑是一个开源项目，欢迎贡献：

1. **提交Issue** — 报告bug、建议新采集器、功能请求
2. **提交PR** — 新采集器脚本、文档改进、bug修复
3. **反馈** — 哪些数据源对你有用、哪些不靠谱

### 如何贡献一个采集器

```python
# 1. 在 scripts/collectors/ 下创建 your_collector.py
# 2. 继承已有的采集器模式
# 3. 在 tianji_runner.py 里注册
# 4. 提交PR
```

> 小提示：优先做免费·无需注册的数据源，零门槛，用户最多。
