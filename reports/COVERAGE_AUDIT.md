## 核心问题

13个采集器看着多，但**大部分是"搜索一下"级别**，不是"持续产出可行动情报"级别。

真正的差距不在"有没有采集器"，在**采集器产出的情报深度**：

```
现在：营销采集器 → 搜索一下竞品名 → 返回几条新闻
需要：营销采集器 → 定时抓FB Ad Library → 解析竞品广告素材/文案/落地页
                    → 定时扫描红人合作 → 更新KOL数据库
                    → 监控广告政策变化 → 触发告警
```

每一个采集器都需要从"扫了一下"升级到"真正干了活"。
